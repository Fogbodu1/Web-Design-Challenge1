Project Summary





We created a python script in this project, where we try to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, we utilized a simple Python library, the OpenWeatherMap API, and data analytics to create a representative model of weather across world cities.

The first requirement was to create a series of scatter plots to showcase the following relationships:
Temperature (F) vs. Latitude
Humidity (%) vs. Latitude
Cloudiness (%) vs. Latitude
Wind Speed (mph) vs. Latitude

The second requirement was to run linear regression on each relationship. This time, separating the plots into Northern Hemisphere (greater than or equal to 0 degrees latitude) and Southern Hemisphere (less than 0 degrees latitude). Then using the list of cities and OpenWeatherMap API to get the weather. MatPlotLib was used to plot the data with the latitude on the appropriate graphs using each city's maximum temperature, humidity, cloudiness, or wind speed.
